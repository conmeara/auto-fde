# gstack

Source: https://github.com/garrytan/gstack

## Why It Matters For Auto-FDE

Garry Tan's gstack is a reference for an opinionated agent setup with structured software-development roles and commands.

Use it as inspiration for:

- specialist FDE roles such as workflow discoverer, planner, reviewer, release lead, and QA
- command-shaped team workflows
- host-aware setup patterns
- making role boundaries legible enough for users to invoke deliberately
