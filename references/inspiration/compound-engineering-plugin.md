# Every Compound Engineering Plugin

Source: https://github.com/EveryInc/compound-engineering-plugin

## Why It Matters For Auto-FDE

Every's Compound Engineering plugin is a reference for packaging an engineering workflow as a reusable agent plugin across multiple coding-agent hosts.

Use it as inspiration for:

- lifecycle commands and skills that compound across projects
- multi-host install and distribution patterns
- planning, implementation, review, and postmortem behaviors
- documenting how a team should adopt a plugin, not only how to install it
