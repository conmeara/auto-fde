Creative Studio

A Claude Code plugin for the Creative Studio at Deloitte. It turns the studio's production process into skills Claude can run — briefs, budgets, call sheets, cut reviews, delivery — working from the studio's own templates and connected to Airtable, Frame.io, and Figma.

How it works

Describe the task in plain language and the right skill loads — with the studio's real template, reference data, and checklist for that job. Skills are organized by production phase, each phase has a /cs-* command, and /cs shows where a project sits and what runs next. Anything client-facing waits on your approval.

The skills

What's in the plugin

One skill per task, organized by phase. PM is production management — scheduling, budget, logistics, comms. Creative is the making — brief, script, edit, design.
